Resource Profile: ClinicalImpressionEncouClinicalImpressionnter

Official URL: http://example.com/fhir/example/StructureDefinition/ClinicalImpressionEncounter Version: 1.0.0
Active as of 2024-10-08 Computable Name: ClinicalImpressionEncounter

Este perfil representa as restrições aplicadas ao recurso ClinicalImpression pelo Guia de Implementação FHIR do Sistema Clínico de Triagem e Aconselhamento (TCCS). Descreve as expectativas mínimas para o recurso ClinicalImpression quando utilizado na composição do sistema ou em um dos recursos referidos.

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
